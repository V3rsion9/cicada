This is a demo for demonstrating data collection using the push method.

Connect using telnet on 6000 port and send strings like #ip #var #value.
Send 192.168.0.23 CPU 50 and 192.168.0.23 TEMP 32 to set CPU and TEMP to 50 and 10.
Send 192.168.0.23 CPU 60 to override current value of CPU.

Use CPU and TEMP as address.



The polling period should be less than the rate of change of data.
