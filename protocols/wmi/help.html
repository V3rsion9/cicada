Use @N to get N-row in result, e.g. alias: logicaldisk, property: Name@1.

Use extended alias to prefilter result, e.g. alias: service where (caption = 'SNMP Service'), property: state to get current state of SNMP service.

Use @count to get row count in result, e.g. alias: service where (state = 'Running'), property: caption@count to get running service count.


You can get any performance counter via alias path some_perf_class.See example below:
alias: path Win32_PerfRawData_Tcpip_NetworkInterface where packetspersec!=0, property: bytesreceivedpersec@{INDEX}

To get available classes and counters follow next steps:

  • Run %windir%\system32\wbem\wbemtest.exe (info)
  • Push Connect and input root\cimv2
  • Push Open Class... and input Win32_PerfRawData (or any known class).
  • Push Derived. Select required class by double click.
  • Set Hide system properties flag and below you will see available props.
