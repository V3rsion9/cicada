Use square brackets to specify protocol (http or https) and port eg

  • /some-url?id=5 - get http data used 80 port (default)
  • [555]/some-url?id=5 - get http data used 555 port
  • [https]/some-url?id=5 - get https data used 443 port (default)
  • [https:500]/some-url?id=5 - get https data used 500 port


Special selectors

  • @time - response time, ns
  • @code - http status code
  • @size - content size
  • @expires (https only) - the number of days before the SSL certificate expires.


When type is text then selector is regular expression.
Example: use (\d+) for Hello 123 world! text to get 123.


When type is json then selector is js-expression. 

// Example json
{
	"a": "A",
	"b": [10, 20],
	"c": {
		"x": 30, 
		"y": 40
	},
	"d": [
		{"name": "AAA", "num": 123},
		{"name": "BBB", "num": 124},
	]
}

// Selector => result
a => "A"
b[1] => 20
c.x => 30
d.filter((e) => e.num == 124).name => BBB


When type is xml then receiving xml converted to json-object via fast-xml-parser. Selector is js-expression.
