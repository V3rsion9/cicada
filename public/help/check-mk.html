Sections vary depending on the operating system.You can run check_mk_agent test on client machine to get supported sections.

Use A[[[B]]] as section, e.g. logwatch[[[System]]], to get subset of section. Don't set the pattern to get all section data.

Below some sections with description and pattern example.

  • check_mk - agent version and parameters.
    Version: Version: (.*)
    OS: AgentOS: (.*)
  • uptime - system uptime.
    Uptime: (.*)
  • df - size and usage of all fixed disks.
    Total size of disk C, kb: ^C:(?:[\D]*)([\d]*)
    Usage size of disk C, kb: ^C:(?:[\D]*)(?:[\d]*) ([\d]*)
  • ps:* - All running processes
  • services - all running and stopped services
    Check-mk agent status: Check_MK_Agent (running|stopped)
  • mem - usage of RAM and the page file.
    Free memory, kB: MemFree(?:[\D]*)([\d]*)
  • winperf_if, winperf_phydisk, winperf_processor - Windows specific
  • fileinfo:*
  • logwatch - eventlogs found in the system
